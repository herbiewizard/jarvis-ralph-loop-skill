# AGENTS.md

## Validation Backpressure
- Test: npm test
- Lint: npm run lint
- Typecheck: npm run typecheck
- Build: npm run build

## Rules
- If any check fails, fix before commit.
- Keep changes scoped to selected task.
- Log discoveries in IMPLEMENTATION_PLAN.md.

## Safety
- Ask before destructive operations.
- Never commit secrets.
